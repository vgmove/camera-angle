# Camera Angle
<div align="center">
  <img src=".meta/preview.jpg" width="800"/> <br>
</div>

Blender3D addon for quick setup of the orthographic camera angle to achieve a specific projection type, with the ability to change angles.

The rotation target is an empty parent object, which can be created manually or automatically from the addon panel.
<img src=".meta/preview_2.png" width="300"/> <br>

Rotation occurs horizontally or vertically, with a step corresponding to the selected projection type. <br>
Supported projections:
- Isometric
- Trimetric
- Dimetric
